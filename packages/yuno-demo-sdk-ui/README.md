# Yuno Payment SDK

A lightweight TypeScript SDK for integrating payment processing secure

## Installation

```sh
npm install yuno-demo-sdk-ui
```

## Example

```typescript
import YunoIframe from "yuno-demo-sdk-ui";

export default function Page() {
  return (
    <div>
      <YunoIframe url="https://url.secure.y.uno" height="800px" />
    </div>
  );
}
```
