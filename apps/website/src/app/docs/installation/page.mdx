# Instalación

Esta guía te ayudará a instalar y configurar Yuno Demo en tu entorno local.

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado:

- **Node.js** 18.x o superior (recomendado: 22.x)
- **npm**, **pnpm**, o **bun** como gestor de paquetes
- **Git** para clonar el repositorio

## Clonar el repositorio

```bash
git clone https://github.com/MrUprizing/demo-yuno.git
cd demo-yuno
```

## Instalar dependencias

### Con npm

```bash
npm install
```

### Con pnpm

```bash
pnpm install
```

### Con bun

```bash
bun install
```

## Configurar variables de entorno

Crea un archivo `.env.local` en el directorio `apps/website`:

```bash
# apps/website/.env.local
NEXT_PUBLIC_API_URL=http://localhost:8080
```

## Iniciar el servidor de desarrollo

```bash
npm run dev
```

La aplicación estará disponible en [http://localhost:3000](http://localhost:3000).

## Estructura del proyecto

```
demo-yuno/
├── apps/
│   ├── backend/          # Backend Java Spring Boot
│   └── website/          # Frontend Next.js
├── packages/
│   └── yuno-demo-sdk/    # SDK compartido
└── ...
```
